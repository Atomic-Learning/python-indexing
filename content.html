Since strings are sequences, you can access individual characters by their position. This is called indexing.

Zero-Based Indexing

In Python, the first character is at index 0, not 1. This is called zero-based indexing:

word = "hello" # 01234 (indices) print(word[0]) print(word[1]) print(word[2]) print(word[3]) print(word[4])

So to get the first character, you use index 0. To get the second character, you use index 1, and so on.

Negative Indexing

Python also lets you use negative indices to count backward from the end of the string. Index -1 refers to the last character, -2 to the second-to-last, and so on:

word = "hello" # -5-4-3-2-1 (negative indices) # 01234 (positive indices) print(word[-1]) print(word[-2]) print(word[-5])

This is useful when you want to get characters near the end without knowing the string's length:

phrase = "Hello back!" print(phrase[-1]) print(phrase[-2]) print(phrase[-11])

A Character Is Still a String

When you use indexing to get a single character, you get back a string that is exactly one character long (not a special "character" type):

word = "hello" first_char = word[0] print(first_char) print(type(first_char))
